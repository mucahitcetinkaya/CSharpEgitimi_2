

VİDEO 39 Boxing İşlemi

object tipinin bütün tipleri kapsayabildiğini bir önceki ders gördük
ve iki adet kod yazdık 
object o1 = 100;
object o2 = 10.90;

ve bunların kutulama yani boxing oldugundan bahsedeceğiz dedik

object o1 = 100;
object burada diyorki o1 değişkeni içinde 100 değerini saklarım ama bunun tipinde saklarım diyor yani
int olarak saklarım diyor
hangi tip saklama olayı ileri konu


object o2 = 10.90;
object burada diyorki o2 değişkeni içinde 10.90 değerini sonuna bişey yazmadıgımız için
double olarak saklarım diyor
hangi tip saklama olayı ileri konu

object o3 = true; bool olarak da saklıyor

nerede kullanırız
kendi class larımızı yazdıgımız zaman kalıtım konutunda kendi yaptıgımız ana tipe döndürmek isteyeceğiz
orda da bu kutulamayı ve kutudan cıkartmayı kullanacağız
object oranın kapılarını acıyor
class lar konusunda bizim class ımız da object gibi davranabilecek
kendinden sonra kendini kalıtabilecek kalıtım konusunda anlatılacak
